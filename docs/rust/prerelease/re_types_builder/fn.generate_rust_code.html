Expand description

Generates Rust code.

Panics on error.

  • output_crate_path: path to the root of the output crate.

E.g.:

let (objects, arrow_registry) = re_types_builder::generate_lang_agnostic(
    "./definitions",
    "./definitions/rerun/archetypes.fbs",
);
re_types_builder::generate_rust_code(
    ".",
    &objects,
    &arrow_registry,
);
