fn try_to_arrow(&self) -> SerializationResult<Vec<(Field, Box<dyn Array>)>>

Serializes all non-null Components of this Archetype into Arrow arrays.

This can never fail for Rerun’s built-in archetypes. For the non-fallible version, see Archetype::to_arrow.

fn try_from_arrow( data: impl IntoIterator<Item = (Field, Box<dyn Array>)> ) -> DeserializationResult<Self>where Self: Sized,

Given an iterator of Arrow arrays and their respective field metadata, deserializes them into this archetype.

Arrow arrays that are unknown to this Archetype will simply be ignored and a warning logged to stderr.

For the non-fallible version, see Archetype::from_arrow.
